# Reference

Background and prior-art material that informs ensemble's design but is not part of
the product itself.

| Section | What it is |
|---|---|
| [External protocols](external/README.md) | Archived third-party specs for other multiroom / network-audio systems — Snapcast, SlimProto, AirPlay 2, the Roc toolkit, Google Cast, and AES67/PipeWire/PulseAudio RTP — kept verbatim (with provenance) so design decisions can cite concrete prior art. Includes an axis-by-axis comparison with ensemble. |

For ensemble's own protocol and design, see the [architecture docs](../architecture/)
and the [player protocol](../developer/player-protocol.md).
